
# Портфолио

**Стек:** *Pyhton, Go, С*

## Проекты
[sell-helper](https://github.com/automationautomata/sell-helper) – Позволяет пользователю: по штрихкоду или названию товара получить его предзаполненные метаданные (имя, описание, и пр.), посмотреть аналоги, и затем — автоматически опубликовать объявление на маркетплейсе. В качестве системы поиска используется Perplexity. 
Аутентификация по JWT-токену, для кэширования используется Redis, основная БД - Postgres.
Стек: 
- Бекенд: Python, Poetry, FastAPI, Dishka, SQLAlchemy, Alembic, тесты - pytest, 
- Фронтенд: Vue
- Docker

[automata-builder](https://github.com/automationautomata/automata-builder) – десктопное приложение для создания и демонстрации работы автоматов (КДА) – позволяет рисовать их диаграммы переходов, проверять на корректность и рисовать графики.  Стек: Python + PyQt6.

[compressor](https://github.com/automationautomata/compressor) – консольное приложение файла или папки методом Хаффмана. Стек: Go.

[СКУД](https://github.com/automationautomata/SKUD) - система контроля и управления доступом, управляет считывателями карт (работают на ардуино) и политиками их доступа. Стек: Python + asyncio, threading и Tornado, бд - SQLite.
